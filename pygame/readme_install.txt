how to install:

this game needs python (python3, but works also with python2) and pygame installed.

start the game by typing:

python3 pygamerogue.py

or by opening pygamerogue with your preferred python editor and clicking on "run".


how to install python3:

ubuntu, linux:


sudo apt-get install python3

other os:
see http://python.org

how to install pygame for python3:

ubuntu, linux:
see http://askubuntu.com/a/406410/76648
other os:
see https://bitbucket.org/pygame/pygame/downloads
